# Contributing to Inoue

Thanks for helping improve Inoue.

## Ways to contribute
- Report bugs and feature requests
- Improve fingerprinting signatures
- Add support for more technologies or headers
- Improve docs and examples

## Development setup
```bash
git clone https://github.com/alhamrizvi-cloud/Inoue.git
cd Inoue
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Pull request checklist
- Keep changes focused and well documented
- Add or update tests when relevant
- Verify the CLI still runs with `python inoue.py --help`
